6.1 fit_gillnet_dome function

This section demonstrates how to use the fit_gillnet_dome() function to estimate selectivity parameters for gillnet fisheries. The function fits multiple selectivity models to experimental gillnet data and provides comprehensive model comparison capabilities.

The fit_gillnet_dome() function is designed to:

  • Fit multiple selectivity curves to gillnet experimental data
  • Handle both unimodal and bimodal selectivity patterns
  • Provide automatic model comparison and selection
  • Generate plots and statistical summaries
  • Support both automatic and manual starting value specification

The function expects input data with the following structure:

  • Length measurements or frequency data from experimental gillnet fishing
  • Multiple mesh sizes tested simultaneously
  • Data organized in a format compatible with the package’s data processing functions

Loading the package and data:

# Load the package
library(fishLengthAssess)

# Load the example gillnet data
data("raw_data_gillnet")
input_data <- raw_data_gillnet

6.1.1 Example 1

6.1.1.1 Fitting unimodal models only

In many cases, the user may want to start with simpler unimodal selectivity models before considering more complex bimodal patterns. Setting run_bimodal = FALSE fits only the three basic dome-shaped models (i.e., norm.loc: Normal common spread, norm.sca: Normal scaled spread, and lognorm: Lognormal).

# Fit unimodal selectivity models only
result_unimodal <- fit_gillnet_dome(
  input_data = input_data,
  mesh_sizes = mesh_sizes,
  run_bimodal = FALSE,
  output_dir = "model_plots_unimodal",
  length_seq = seq(40, 100, 1)
)

The function automatically generates plots for each fitted selectivity model and saves them in the specified output directory (e.g., model_plots_unimodal).

  binorm.sca = c(55, 4, 65, 4, 3),     # Mode1, SD1, Mode2, SD2, logit(P1)
  bilognorm = c(4, 0.2, 4.2, 0.1, 2)  # log(mu1), log(sigma1), log(mu2), log(sigma2), logit(P1)
)
# Fit models with manual starting values

6.2 GTG dome-shaped LBSPR assessment

6.2.1 Practical Application of GTG Length-Based SPR model including dome-shaped selectivity: Step-by-Step Example

This section demonstrates how to estimate fishing mortality (F/M) and spawning potential ratio (SPR) from catch-at-length data using the GTG dome-shaped length-based SPR model. The selectivity curve is treated as a fixed known input — only F/M is estimated. The selectivity parameters are obtained from fit_gillnet_dome() (see section 5.4) or defined directly by the user.

The example code presented here was adapted from the original implementation (https://github.com/KHommik/DomeShaped_GTG_LBSPR) by Hommik et al. (2020), with modifications to accommodate additional selectivity models and updated simulation routines.

Four examples are provided covering two data formats and two sampling designs:

Example Data type Groups

B Length 5 groups
C Frequency 1 group
D Length 1 group

Four example datasets are included in the package to demonstrate the workflow. Since the data are simulated with known parameters (Linf = 120 cm, K = 0.2, M/K = 1.5, L50 = 60 cm, FM = 1.0, Normal.sca selectivity with SL1 = 55.28 and SL2 = 4.343513^2), the user can verify that the estimated F/M and SPR values are close to the true values used in the simulation.

6.2.1.1 Step 1. Load packages and define common inputs

The life history parameters, length bins, and fleet parameters are the same across all four examples. Define them once here.

Load the required packages and data:

# --- Life history parameters ---
LifeHistoryObj <- new("LifeHistory")
LifeHistoryObj@title        <- "Fish Stock"

LifeHistoryObj@L_units      <- "cm"
LifeHistoryObj@Walpha_units <- "g"
LifeHistoryObj@Linf         <- 120   # asymptotic length (cm)
LifeHistoryObj@K            <- 0.2   # growth rate
LifeHistoryObj@t0           <- 0     # theoretical age at length zero
LifeHistoryObj@L50          <- 60    # length at 50% maturity (cm)
LifeHistoryObj@L95delta     <- 2     # L95 = L50 + L95delta = 62 cm
LifeHistoryObj@MK           <- 1.5   # M/K ratio
LifeHistoryObj@M            <- LifeHistoryObj@MK * LifeHistoryObj@K
LifeHistoryObj@LW_A         <- 0.01  # weight = LW_A x Length ^ LW_B
LifeHistoryObj@LW_B         <- 3
LifeHistoryObj@R0           <- 1E6   # unfished recruitment (does not affect SPR)

# --- Length bins ---
# make_size_bins() automatically builds the correct length range from Linf.
# Change Linc to adjust bin width (default = 1 cm).
SizeBins <- make_size_bins(LifeHistoryObj, Linc = 1)

# --- Fleet parameters: dome-shaped selectivity (fixed) ---
# The selectivity parameters below come from fitting the Normal.sca model
# to the example gillnet data using fit_gillnet_dome() (see Example 1).
# For your own data, replace SL1, SL2, SLmesh, and fishery_mesh with 
# the values from your own selectivity analysis.
#
# IMPORTANT: SL2 must be SQUARED when using Normal.sca.
# select_Millar() returns k2 as a spread coefficient, but 
# GTGDomeLBSPRSim2() expects k2 as a variance coefficient (k2 squared).

FleetPars <- NULL
FleetPars$selectivityCurve <- "Normal.sca"

FleetPars$SLMin            <- NA             # no minimum landing length
FleetPars$use_aggregated   <- FALSE          # use mesh-specific selectivity
FleetPars$fishery_mesh     <- 15.9           # mesh size used in the fishery
FleetPars$FM               <- 1             # starting value for optimization
